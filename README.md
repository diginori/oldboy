http://digiNORI.com , @diginori

### To run:

express@2.2.0 와 socket.io@0.8.2 를 사용함


Install & Run

    	$ npm install express@2.2.0
	$ npm install socket.io@0.8.2

	$ node app.js
	
After running go to this address:

	http://locahost:PORT/
